Curso de Confección y Publicación de Páginas Web

Bienvenido al curso donde aprenderás a crear, diseñar y publicar páginas web desde cero, combinando el uso de herramientas modernas con conocimientos básicos de programación. Este curso está diseñado para principiantes y personas interesadas en tener su propio sitio web personal, profesional o educativo.

¿Qué aprenderás?

  • Cómo construir la estructura de una página web usando HTML.
  • Cómo aplicar estilos y diseños visuales con CSS.
  • Cómo añadir funciones básicas e interactividad con JavaScript.
  • Cómo usar WordPress para crear páginas web sin necesidad de programar.
  • Cómo gestionar y publicar blogs en Blogger.
  • Cómo publicar tu página en Internet para que esté disponible al mundo.

Objetivo del curso

Al finalizar este curso, serás capaz de confeccionar una página web completa, atractiva y funcional, y además publicarla en línea. No importa si prefieres escribir código o usar plataformas visuales, aquí aprenderás ambas formas de hacerlo.

Prácticas realizadas

  • Creación de una página con HTML y CSS.
  • Proyecto interactivo básico con JavaScript.
  • Blog en Blogger con publicaciones reales.
  • Sitio completo en WordPress con páginas, menús y formulario de contacto.

HTML

Creacción de la estructura básica de una pàgina web.

  • Uso de etiquetas semánticas como <header>, <main>, <footer>
  • Creación de formularios de contacto con <form>
  • Construcción de listas, tablas y enlaces

Diseño y estilo de páginas web con colores, tamaños y posicionamiento.

CSS (Hojas de Estilo en Cascada) permite aplicar estilos a los elementos HTML para mejorar su apariencia visual

  • Aplicación de colores , maàrgines y tipografia.
  • Uso de selectores para personalizar elementos.
  • Desiño Responsive con media
  • Creación de animaciones y transicions

JavaScript

Interactividad y dinamismo en páginas web mediante programación.

JavaScript permite controlar el comportamiento de los elementos HTML y responder a acciones del usuario.

  • Manipulación del DOM (Document object Model).
  • Gestión de evento como clics y envíos de formularios.
  • Validaión de furmularios y cálculos en la página.
  • Creación de ventanas modales y cambios dinámicos de continedo.
